# FreeDash 數據分析看板操作手冊

歡迎使用 **FreeDash 數據視覺化分析看板**！本手冊旨在引導您快速上手看板的各項核心功能，包括樞紐彙總資料表、自訂欄位排序與運算、圖表主題同步、以及單一文件離線匯出與部署。

---

## 一、 系統簡介與核心設計
FreeDash 是一款**離線優先 (Offline-First)** 且**無後端 (Client-Only)** 的數據看板生成器：
* **隱私安全**：所有試算表數據（Excel / CSV）均在您的瀏覽器本地解析，絕對不會上傳到任何外部伺服器。
* **高流暢度**：基於 React、Tailwind CSS 及 Chart.js 開發，切換佈局與篩選時均在毫秒級完成。
* **一鍵分享**：可將整個看板設計與數據封裝為單一、全自包含的 `.html` 檔案，下載後可在任何裝置上直接開啟，並完整保留動態篩選器及圖表互動。

---

## 二、 樞紐彙總資料表 (Pivot Table)
資料表卡片（Data Card）具備如同 Microsoft Excel 樞紐分析表及 Power Query 般強大的分組彙總、多層樹狀摺疊與資料清洗能力。

### 1. 啟用多層階層分組 (Tree Table / Multi-level Grouping)
* **分組模式切換**：在卡片編輯面板的「分組彙總模式」下拉選單中：
  * **原始明細資料 (Raw Data)**：直接列出每一筆原始明細。
  * **啟用分組彙總 (Grouped / Pivot)**：開啟分組階層設定。
* **多層分組階層 (Multi-level Hierarchy)**：您可以新增複數個欄位進行嵌套分組：
  * 點擊「新增分組階層欄位」下拉選單選取欲加入的欄位。
  * 透過各分組欄位右側的 `▲` (提升階層) 及 `▼` (降低階層) 按鈕，可任意配置分組階層上下順序（例如：`L1 部門` -> `L2 職稱`）。
  * 點擊 `❌` 可移除該分組欄位。
* **摺疊與展開控制 (Toggle Expand/Collapse)**：
  * 啟用多層分組後，資料表會以樹狀結構 (Tree Table) 呈現，子群組預設為展開狀態。
  * 在群組首欄的欄位值旁，點擊 `▼` (摺疊) 可以收合該群組下的所有子階層，點擊 `▶` (展開) 即可重新開啟。
  * 此動態展開/摺疊功能在**匯出為離線 HTML 後仍然 100% 完整保留並可操作**。

### 2. 區間分群維度 (Group Intervals)
在階層分組欄位清單中，針對特定類型的欄位，系統會顯示相應維度選單：
* **日期欄位 (Date)**：可選「按週 (Weekly)」、「按月 (Monthly)」或「按年 (Yearly)」，系統會自動將日期轉換格式並依相應時間段分組。
* **數值欄位 (Number)**：可選「自動劃分 5 個等距區間」，系統會自動計算最大值與最小值，並等距切分為五個數字範圍（如 `10,000 - 30,000` 等）。

### 3. 呈現欄位勾選與自訂排序
在編輯面板的「呈現的資料欄位勾選與排序」區塊中：
* **分組欄位鎖定**：已設定為「階層分組」的欄位會標示黃色的 `分組` 標籤，並自動鎖定在表格最左側的前幾欄，核取方塊會自動鎖定為勾選狀態，避免欄位結構混亂。
* **呈現順序調整**：非分組的欄位，您可以透過勾選與否決定顯示狀態，並使用 `▲` (向上移) 及 `▼` (向下移) 按鈕自由排列順序。

### 4. 欄位彙總計算與小計 (Aggregations & Subtotals)
* **計算選單**：非群組欄位的右側會出現計算下拉選單：
  * **無 (去重字串)**：預設，將群組內的所有字串以逗號連接（超過 3 項顯示「... 等 N 項」）。
  * **加總 (SUM)**：對該欄位之數值進行加總。
  * **計數 (COUNT)**：計算該群組內的總筆數。
  * **平均 (AVG)**：計算該群組內數值的平均值。
* **底線小計列 (Subtotals Row)**：當表格資料列大於 0 且啟用分組彙總時，表格最後一列會自動插入一筆**「小計 (Total)」**，彙總整張表格的所有指標。

## 三、 逆樞紐資料清洗 (Power Query Unpivot / 寬轉長)
當您上傳的原始試算表格式不符合資料庫正規化設計（例如：前面幾欄是基本資料，後面是一堆以月份「1月」、「2月」或部門「BU1」、「BU2」作為獨立欄位的矩陣表格），您可以使用本工具在匯入時進行資料清洗。

### 1. 執行步驟
1. 切換至「上傳資料」頁面。
2. 點擊表格右上方橘色的 **「逆樞紐資料清洗 (Unpivot)」** 按鈕。
3. 在彈出視窗中設定：
   * **新屬性欄位名稱**：例如輸入「月份」或「部門」。
   * **新數值欄位名稱**：例如輸入「銷售額」或「工時」。
   * **勾選合併欄位**：在下方勾選您要「摺疊」合併的所有欄位（例如勾選 1月、2月... 12月）。
4. 點擊 **「確認執行轉換」** 即可。

### 2. 轉換效果
系統會將原始的 1 筆橫向矩陣資料拆解為多筆直向的明細資料，並自動轉換所有對應欄位的型態。完成後，您即可利用資料表卡的「多層分組階層」或圖表卡，輕鬆地針對「月份」或「部門」進行彙總加總或折線趨勢圖分析！

## 三、 圖表卡片與主題同步 (Charts)
FreeDash 提供長條圖、折線圖、圓餅圖、進度環 (Progress Ring) 及疊加對比圖 (Bar Overlap) 等多種圖表卡片。

### 1. 毫秒級主題切換與格線自適應
* **格線修正**：圖表內嵌了動態銷毀與重建機制。當您在系統右上角切換「淺色 (Light)」與「深色 (Dark)」模式或變更顏色範本時，圖表會立即重新繪製。
* **護眼對比度**：在深色模式下，格線會自動切換為低對比度的半透明灰藍色 (`rgba(51,65,85,0.4)`)，且文字標籤會切換為清晰防眩光的粉灰色；淺色模式下則切換為柔和的淡灰色 (`rgba(226,232,240,0.7)`)，保證長期觀看舒適。

### 2. 疊加對比圖 (Bar Overlap)
* 適用於「實際值與目標值」的對比。
* 目標值（Plan）會以寬而淺的灰色長條在後方展示，實際值（Actual）會以窄而亮的主題色在前方疊加。若設定了閾值（Thresholds），實際值長條的顏色將依據達標率自動改變（如低於 60% 變紅色、超過 100% 變綠色）。

## 四、 離線自包含 HTML 匯出 (Export)
* 點擊右上角「匯出離線 HTML 儀表板」按鈕，系統會將所有設定、過濾器狀態、圖表配置及當前載入的數據封裝成一個自包含的 `.html` 文件。
* **100% 相同佈局與外觀**：匯出的 HTML 檔案排版比例、圖表刻度、文字大小、格線顏色等外觀細節，均與您在 FreeDash 線上看到的完全一致，並搭載了完全相同的 Chart.js v4 渲染引擎及小計彙總運算器。
